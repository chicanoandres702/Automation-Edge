# **App Name**: Automaton Edge

## Core Features:

- Natural Language Prompt Input: An intuitive, mobile-optimized input field for users to describe desired browser automation tasks using everyday language.
- AI Automation Core: An AI-powered tool that interprets natural language prompts to create and execute complex browser automation workflows, particularly for survey completion and anti-disqualification strategies.
- Real-time Action Visualizer: Displays the AI agent's current activity and predicted next steps in real-time within the sidebar, ensuring transparency during automation, with adaptive display for mobile screens.
- Automation History Log: Maintains a chronological record of all executed automation tasks, their outcomes, and any specific notes or alerts, with options for both local and cloud-synced persistence.
- Agent Control Panel: Dedicated UI elements, designed for both desktop and touch interfaces, to start, pause, or completely halt the AI automation agent's operation at any point.
- Contextual Awareness Tool: An AI-powered tool that analyzes the current web page content (e.g., survey questions) to inform its automation actions, preventing common disqualifications by adhering to survey logic.
- Secure Google Authentication: Enables users to securely log in and manage their profiles using their existing Google accounts, integrating with backend services, and providing a consistent experience across devices.
- Flexible Data Persistence (Firestore & Local): Manages the storage of user settings, automation configurations, and history. Supports client-side local storage for standalone operation and optional synchronization with Google Cloud Firestore for cross-device access and enhanced data integrity.
- Mobile-Optimized Functionality: Ensures the full operational capability of the AI agent and all its features are accessible and robust on mobile browsers like Android Edge, maintaining performance and usability for continuous, long-duration tasks.

## Style Guidelines:

- Background color: A deep, rich charcoal (#1E1E1E) for a sophisticated and modern dark mode foundation.
- Primary color: A refined, muted cerulean blue (#5C85A8) for interactive elements and key information, providing a calm yet clear contrast.
- Accent color: A sharp, energetic neon green (#43F91F) used sparingly for critical feedback, active states, and subtle highlights, adding a futuristic touch.
- Body and headline font: 'Space Grotesk' (sans-serif) for its modern, techy, and clean appearance, suitable for both concise headings and brief informational text within the sidebar on all screen sizes.
- Utilize minimalist, crisp geometric icons that align with the futuristic and efficient aesthetic, using the neon green accent for active states, optimized for clarity on smaller mobile screens.
- Implement a clear and segmented sidebar layout with distinct sections for prompt input, real-time feedback, and control elements, ensuring intuitive navigation within a compact space for a professional feel. The layout must be fully responsive, adapting seamlessly to various screen sizes and orientations, especially for mobile devices.
- Incorporate subtle, fast, and smooth transitions for state changes (e.g., play/pause), progress indicators, and visual feedback on AI actions to enhance the sense of responsiveness and advanced technology, ensuring smooth performance on mobile.
